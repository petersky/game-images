Prepare — shift

API keys

Advanced: paste token manually

Sign in uses the ChatGPT/Codex OAuth flow (browser login, callback on port 1455). API key env var still takes precedence. If port 1455 is in use, stop Codex/other tools or set GAME_IMAGES_OPENAI_USE_APP_REDIRECT=1.
